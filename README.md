# 🚀 Vienna Project

Vienna – your all-in-one tool. Simple to build and run on any system.

---

## 👍 How to Use
start start of you system

```bash
bash start.sh
```
```batch
start.bat
```
## 🛠 How to Build

Build from source using Maven:

```bash
export MAVEN_OPTS="-Dhttps.protocols=TLSv1.2"
bash mvnw clean install
```
or go ![releases](https://github.com/NEXTDROIDER/Vienna-Update/releases)
